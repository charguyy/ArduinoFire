# ArduinoFire
A basic fire effect for an LED strip. For use with an Arduino.

# Materials

1x Arduino (Nano, Uno, or Mega)
1x 220 ohm resistor (optional)
1x 470uF 16V capacitor (optional)
1x RGB LED strip (I used a DotStar strip)
1x 5V power supply

Wire
Soldering iron
